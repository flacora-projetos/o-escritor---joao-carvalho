# João Carvalho · Mockup de landing page

Protótipo estático criado para apresentar a identidade visual "gótico neoclássico" do escritor João Carvalho.

- Abra `index.html` em um navegador (ou configure o diretório como root de um site estático no Vercel) para visualizar o mockup completo.
- As imagens estão em `assets/` e já estão referenciadas pelo HTML com caminhos relativos.
- Qualquer ajuste de copy, datas ou novos ativos pode ser feito diretamente neste arquivo antes de publicar.
